# Visualiser les patterns d’usage et de bien-être associés aux réseaux sociaux

Projet de dataviz (M2 Lyon 1) sur l’usage des réseaux sociaux (TikTok, Instagram, X, etc.)
et leur lien avec le bien-être (stress, anxiété, sommeil).

- Dataset 1 : Kaggle — Mental Health and Social Media Balance
- Dataset 2 : Questionnaire — Usage des réseaux sociaux et impact sur le bien-être

Structure :
- `doc/` : cadrage du projet
- `data/` : données brutes (`raw/`) et nettoyées (`processed/`)
- `app/` : page web + scripts D3.js
